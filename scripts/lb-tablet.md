# 💊 LB Tablet

**Version :** lb-tablet 1.6.0

{% hint style="info" %}
LB Tablet est la tablette utilisée principalement par les métiers (Police, EMS, Staff). Elle donne accès à des apps professionnelles.
{% endhint %}

---

## ✨ Applications disponibles
- 👮 MDT Police (wasabi_police)
- 🚑 MDT EMS (wasabi_ambulance)
- 📰 Weazel News (app staff)
- 🏢 Boss Menu (gestion entreprise)

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — La tablette s'utilise avec `/tablet` ou via l'item dans l'inventaire.
{% endhint %}

{% hint style="success" %}
**Tip 2** — Le **Boss Menu** (`boss_menu`) est accessible depuis la tablette pour les chefs d'entreprise — gestion des employés, salaires, coffre société.
{% endhint %}

{% hint style="warning" %}
**Attention** — Certaines apps de la tablette nécessitent d'être **en service** (`/duty`) pour être accessibles.
{% endhint %}
