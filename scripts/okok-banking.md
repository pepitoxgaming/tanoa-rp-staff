# 🏦 okok Banking

**Version :** okokBanking 1.0.2

{% hint style="info" %}
okok Banking gère le système bancaire du serveur : comptes personnels, comptes société, virements et historique des transactions.
{% endhint %}

---

## ✨ Fonctionnalités
- Comptes bancaires personnels
- Comptes société (boss menu)
- Virements entre joueurs
- Historique des transactions
- ATM (distributeurs)
- Facturation (okokBilling)

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — En cas de litige financier, les transactions sont **loggées** — demande au Développeur de consulter les logs BDD.
{% endhint %}

{% hint style="success" %}
**Tip 2** — `okokBilling` permet aux entreprises (police, hôpital...) d'envoyer des factures aux joueurs directement sur leur téléphone.
{% endhint %}

{% hint style="success" %}
**Tip 3** — Si un joueur signale un bug de banque (argent disparu), vérifie d'abord avec `/givemoney` si c'est un problème d'affichage ou réel.
{% endhint %}

{% hint style="warning" %}
**Attention** — Ne jamais donner de l'argent à un joueur sans justification valable. Tout don non justifié est considéré comme de l'abus de pouvoir.
{% endhint %}
