# 🔫 Wasabi Police

**Version :** wasabi_police 1.10.8

{% hint style="info" %}
Wasabi Police est le script de métier policier du serveur. Il gère les arrestations, menottes, MDT, armureries et bien plus.
{% endhint %}

---

## ✨ Fonctionnalités
- Système d'arrestation et de menottes
- MDT (Mobile Data Terminal) via tablette
- Armurerie police
- Radar de vitesse
- Gestion des preuves (wasabi_evidence)
- Hélicam (0r-helicam)

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — Si un policier est bloqué avec des menottes buggées, utilise `/freeze [id]` puis `/freeze [id]` à nouveau pour reset l'état.
{% endhint %}

{% hint style="success" %}
**Tip 2** — Les preuves collectées sont gérées par `wasabi_evidence` — elles apparaissent dans le MDT de la tablette.
{% endhint %}

{% hint style="success" %}
**Tip 3** — L'hélicam s'active avec `/helicam` depuis un hélicoptère police — utile pour les poursuites et la surveillance.
{% endhint %}

{% hint style="warning" %}
**Attention** — Si un joueur abuse du système d'arrestation (menottes abusives, arrestations non-RP), c'est sanctionnable via `/warn`.
{% endhint %}
