# 🏪 JG Dealership

**Version :** jg-dealerships v2.1.0

{% hint style="info" %}
JG Dealership gère les concessionnaires automobiles du serveur. Les joueurs peuvent acheter, financer et tester des véhicules.
{% endhint %}

---

## ✨ Fonctionnalités
- Achat de véhicules en jeu
- Système de financement
- Test drive
- Gestion du stock par le vendeur
- Interface moderne

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — Les prix des véhicules sont configurés dans les fichiers du script. Pour modifier un prix, contacte le **Développeur Chef**.
{% endhint %}

{% hint style="success" %}
**Tip 2** — Si un joueur a payé mais n'a pas reçu son véhicule, vérifie les logs et utilise `/giveitem` ou `/admincar` pour lui donner manuellement.
{% endhint %}

{% hint style="warning" %}
**Attention** — Les véhicules achetés sont enregistrés en BDD. Ne pas utiliser `/dvdb` sur un véhicule acheté sans accord du joueur.
{% endhint %}
