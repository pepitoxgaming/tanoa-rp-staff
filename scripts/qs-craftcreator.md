# ⚒️ QS Craft Creator

**Version :** qs-crafting 1.0.35

{% hint style="info" %}
QS Craft Creator gère le système de craft du serveur. Il permet de définir des recettes de fabrication pour les items et armes.
{% endhint %}

---

## ✨ Fonctionnalités
- Recettes de craft personnalisables
- Interface visuelle intuitive
- Intégration complète avec QS Inventory
- Support des tables de craft (établis, labos...)

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — Les recettes de craft sont accessibles via `/drugcreator` pour les drogues et les postes de craft définis par le dev.
{% endhint %}

{% hint style="success" %}
**Tip 2** — Si un joueur ne peut pas crafter, vérifie qu'il a **tous les ingrédients** ET qu'il est **au bon endroit** (zone de craft).
{% endhint %}

{% hint style="warning" %}
**Attention** — Les modifications de recettes nécessitent un accès développeur. Contacte le **Développeur Chef** pour toute demande de modification.
{% endhint %}
