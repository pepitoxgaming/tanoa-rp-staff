# 🔑 QS Vehicle Keys

**Version :** qs-vehiclekeys 4.0.29

{% hint style="info" %}
QS Vehicle Keys gère le système de clés de véhicules. Chaque véhicule possède des clés uniques qui peuvent être données, volées ou dupliquées.
{% endhint %}

---

## ✨ Fonctionnalités
- Clés uniques par véhicule
- Partage de clés entre joueurs
- Système de vol de clés
- Intégration avec les garages JG

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — Si un joueur est bloqué hors de son véhicule, utilise `/giveitem [id] [vehicle_key] 1` ou spawn-lui un nouveau véhicule.
{% endhint %}

{% hint style="success" %}
**Tip 2** — Les clés sont liées à la **plaque du véhicule**. Si la plaque change (`/vplate`), les anciennes clés deviennent invalides.
{% endhint %}

{% hint style="warning" %}
**Attention** — En cas de bug de clés, un redémarrage du script `qs-vehiclekeys` peut résoudre la plupart des problèmes.
{% endhint %}
