# 🛒 QS Shop Creator

**Version :** qs-shops-creator 1.1.59

{% hint style="info" %}
Shop Creator permet de créer et gérer des boutiques en jeu sans modifier les fichiers de config.
{% endhint %}

---

## ✨ Fonctionnalités
- Création de boutiques en jeu via commande
- Gestion des prix et stocks
- Support multi-frameworks (ESX/QB)
- Intégration Quasar Inventory

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — Utilise `/drugcreator` pour créer des points de vente de drogues directement en jeu, sans toucher aux fichiers de config.
{% endhint %}

{% hint style="success" %}
**Tip 2** — Les boutiques créées en jeu sont sauvegardées en **base de données** — elles persistent après redémarrage du serveur.
{% endhint %}

{% hint style="warning" %}
**Attention** — `/delete_shop` ne fonctionne que sur les boutiques créées **en jeu**. Les boutiques en config ne peuvent pas être supprimées via commande.
{% endhint %}

---

## ⌨️ Commandes utiles

| Commande | Description |
|----------|-------------|
| `/drugcreator` | Créer points de vente drogues |
| `/jobcreator` | Créer des métiers |
| `/blipscreator` | Gérer les blips sur la carte |
| `/create_shop` | Créer une boutique |
| `/delete_shop` | Supprimer une boutique (en jeu uniquement) |
