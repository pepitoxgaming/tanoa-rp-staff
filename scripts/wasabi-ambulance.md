# 🚑 Wasabi Ambulance

**Version :** wasabi_ambulance_v2 2.0.7

{% hint style="info" %}
Wasabi Ambulance est le script de métier médical du serveur. Il gère les soins, le MDT EMS, les civières et le système de mort.
{% endhint %}

---

## ✨ Fonctionnalités
- Soins des joueurs blessés
- MDT EMS via tablette
- Système de civière
- Défibrillateur
- AI EMS en cas d'absence de médecins (wp-ai-ems)
- Système béquilles (wasabi_crutch)

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — `/revive [id]` est la commande staff pour ranimer un joueur rapidement sans passer par les animations EMS.
{% endhint %}

{% hint style="success" %}
**Tip 2** — Si aucun médecin n'est en service, le script `wp-ai-ems` prend le relais automatiquement avec des médecins NPC.
{% endhint %}

{% hint style="success" %}
**Tip 3** — Un joueur en état critique peut être transporté avec la civière — il faut que l'EMS soit proche et en service.
{% endhint %}

{% hint style="warning" %}
**Attention** — Ne pas abuser de `/revive` sans contexte RP. Les joueurs doivent interagir avec les EMS en jeu si possible.
{% endhint %}
