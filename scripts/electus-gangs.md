# 🏘️ Electus Gangs

**Version :** electus_gangs 2.0.8

{% hint style="info" %}
Electus Gangs gère le système de gangs du serveur : territoires, membres, grades et guerres de territoires.
{% endhint %}

---

## ✨ Fonctionnalités
- Création et gestion de gangs
- Système de territoires
- Grades personnalisables
- Intégration avec les labos illégaux
- Statistiques de gang

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — `/manage_gangs` permet aux admins de créer, modifier ou supprimer des gangs directement en jeu.
{% endhint %}

{% hint style="success" %}
**Tip 2** — `/manage_zones` permet de définir et modifier les zones de territoire contrôlées par les gangs.
{% endhint %}

{% hint style="success" %}
**Tip 3** — En cas de conflit entre gangs, les zones contestées sont visibles en temps réel sur la map via les blips.
{% endhint %}

{% hint style="warning" %}
**Attention** — La suppression d'un gang via `/manage_gangs` est **irréversible**. Tous les membres perdront leur affiliation.
{% endhint %}

---

## ⌨️ Commandes utiles

| Commande | Description | Accès |
|----------|-------------|-------|
| `/gang_menu` | Menu du gang | Membres |
| `/manage_gangs` | Gérer les gangs | Admin |
| `/manage_zones` | Gérer les zones | Admin |
