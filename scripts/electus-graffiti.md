# 🎨 Electus Graffiti

**Version :** electus_graffiti 1.2.4

{% hint style="info" %}
Electus Graffiti permet aux joueurs de tagger des zones en jeu. Les gangs peuvent revendiquer des territoires via les graffitis.
{% endhint %}

---

## ✨ Fonctionnalités
- Tags personnalisés par gang
- Système de revendication de territoire
- Graffitis persistants
- Interaction avec le système de gangs

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — Les graffitis sont liés aux gangs — un tag posé par un gang marque visuellement leur territoire.
{% endhint %}

{% hint style="success" %}
**Tip 2** — En cas de graffiti abusif ou non-RP, tu peux supprimer le tag via le menu admin.
{% endhint %}

{% hint style="warning" %}
**Attention** — Les graffitis sont des éléments RP importants. Ne pas les supprimer sans raison valable.
{% endhint %}
