# 🏠 QS Housing

**Version :** qs-housing 5.3.14

{% hint style="info" %}
QS Housing gère le système immobilier du serveur. Les joueurs peuvent acheter, louer et meubler des propriétés.
{% endhint %}

---

## ✨ Fonctionnalités
- Achat et vente de propriétés
- Système de location
- Ameublement personnalisé
- Coffre privé dans chaque maison
- Garages privés liés

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — `/houseMenu` permet au propriétaire de gérer sa maison (meubles, stockage, paramètres).
{% endhint %}

{% hint style="success" %}
**Tip 2** — `/realestate` ouvre le menu immobilier pour voir toutes les propriétés disponibles.
{% endhint %}

{% hint style="success" %}
**Tip 3** — `/hireRenter` (admin) permet de retirer un locataire d'une propriété en cas de litige.
{% endhint %}

{% hint style="warning" %}
**Attention** — `/resetentrycoords` remet toutes les coordonnées de portes à défaut. À utiliser uniquement si plusieurs maisons ont des portes buggées.
{% endhint %}

---

## ⌨️ Commandes utiles

| Commande | Description | Accès |
|----------|-------------|-------|
| `/houseMenu` | Menu maison | Propriétaire |
| `/hireRenter` | Retirer locataire | Admin |
| `/realestate` | Menu immobilier | Staff |
| `/resetentrycoords` | Reset coordonnées portes | Admin |
