# 🎒 Quasar Inventory

**Version :** qs-inventory 3.7.17

{% hint style="info" %}
Quasar Inventory est le système d'inventaire principal du serveur. Il gère tous les items, armes, et interactions entre joueurs.
{% endhint %}

---

## ✨ Fonctionnalités principales
- Inventaire visuel drag & drop
- Gestion des armes et munitions
- Hotbar (barre d'accès rapide)
- Fouille de joueurs
- Coffres, véhicules, caisses

---

## 💡 Tips Staff

{% hint style="success" %}
**Tip 1** — Utilise `/openinventorytarget [id]` pour voir l'inventaire d'un joueur sans qu'il le sache. Utile pour les contrôles discrets.
{% endhint %}

{% hint style="success" %}
**Tip 2** — `/giveitem [id] [item] [count]` : les noms d'items sont en **minuscules avec underscores**. Ex: `weapon_pistol`, `bread`, `water`.
{% endhint %}

{% hint style="success" %}
**Tip 3** — Si un joueur a son inventaire bloqué/figé, utilise `/resetinv [id]` pour le réinitialiser.
{% endhint %}

{% hint style="warning" %}
**Attention** — `/clearinv [id]` supprime **définitivement** tout l'inventaire. Toujours confirmer avant d'utiliser.
{% endhint %}

---

## ⌨️ Commandes utiles

| Commande | Description |
|----------|-------------|
| `/inventory` | Ouvre l'inventaire |
| `/hotbar` | Ouvre la barre rapide |
| `/giveitem [id] [item] [count]` | Donner un item |
| `/giveweapon [id] [weapon] [ammo]` | Donner une arme |
| `/clearinv [id]` | Vider l'inventaire |
| `/resetinv [id]` | Réinitialiser l'inventaire |
| `/openinventorytarget [id]` | Voir l'inventaire d'un joueur |
| `/admin_giveitem` | Menu interactif de don d'items |
