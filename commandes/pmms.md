# 🎵 PMMS — Lecteur Multimédia

| Commande | Description |
|----------|-------------|
| `/pmms` | Ouvre le panneau de contrôle |
| `/pmms_play [url]` | Lire une musique ou vidéo sur le lecteur le plus proche |
| `/pmms_pause` | Mettre en pause la lecture |
| `/pmms_stop` | Arrêter la lecture |
| `/pmms_status` | Afficher l'état du lecteur le plus proche |
| `/pmms_presets` | Liste des musiques/vidéos préréglées |
| `/pmms_vol [volume]` | Définir votre volume personnel de base |
| `/pmms_fix` | Réinitialiser le client (corrige certains problèmes audio/visuels) |
| `/pmms_ctl` | Contrôle avancé des lecteurs multimédias |
| `/pmms_add` | Ajouter ou modifier un modèle de lecteur prédéfini |
| `/pmms_refresh_perms` | Actualiser les permissions pour tous les clients |
