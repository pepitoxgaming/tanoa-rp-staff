# 🎒 Inventaire

| Commande | Description | Accès |
|----------|-------------|-------|
| `/closeinv` | Ferme instantanément l'inventaire | Staff |
| `/inventory` | Ouvre instantanément l'inventaire | Staff |
| `/hotbar` | Ouvre instantanément la barre d'accès rapide | Staff |
| `/handsup` | Lève les mains et active le vol depuis l'inventaire | Tous |
| `/searchplayer` | Fouille le joueur le plus proche | Staff |
| `/randomitems` | Donne une quantité aléatoire d'objets (test) | Admins |
| `/resetinv [id]` | Réinitialise l'inventaire du joueur | Admins |
| `/clearinv [id]` | Vide complètement l'inventaire du joueur | Admins |
| `/giveitem [id] [item] [count]` | Donne des objets à un joueur | Admins |
| `/giveweapon [id] [weapon] [ammo]` | Donne des armes à un joueur | Admins |
| `/repairweapon [id]` | Répare toutes les armes du joueur (100% durabilité) | Admins |
| `/openinventorytarget [id]` | Consulte l'inventaire d'un joueur cible | Admins |
| `/admin_giveitem` | Menu interactif pour donner des objets | Admins |
