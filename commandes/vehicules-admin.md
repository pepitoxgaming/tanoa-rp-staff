# 🚗 Commandes Véhicules — Administration

| Commande | Description | Accès |
|----------|-------------|-------|
| `/iv` | 🔒 Mettre un véhicule en fourrière | Police (config) |
| `/privategarages` | 🏠 Créer un garage privé | Immobilier (config) |
| `/setjobvehicle [job] [grade]` | 🛠️ Déplacer un véhicule vers un garage de métier | Admins |
| `/removejobvehicle [id]` | ↩️ Rendre le véhicule au propriétaire | Admins |
| `/setgangvehicle [gang] [grade]` | 👥 Déplacer vers garage de gang (QBCore) | Admins |
| `/removegangvehicle [id]` | ↩️ Rendre le véhicule (gang) au propriétaire | Admins |
| `/vplate` | 🔢 Changer la plaque d'un véhicule | Admins |
| `/dvdb` | 🗑️ Supprimer un véhicule de la BDD | Admins |
| `/admincar` | ➕ Ajouter le véhicule actuel au garage (ESX) | Admins |
| `/vreturn [plaque]` | 🚘 Rentrer les véhicules laissés à l'extérieur | Staff |
