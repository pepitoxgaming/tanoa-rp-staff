# 🛠️ Scripts Jaksam

| Commande | Description |
|----------|-------------|
| `/robberycreator` | 💰 Ouvre le panel de création et gestion des braquages (magasins, banques, bijoux...) |
| `/jobcreator` | 📂 Crée des métiers, gère les grades, vestiaires, garages et coffres d'entreprise |
| `/blipscreator` | 📍 Gestion des points sur la carte. Ajoute, modifie ou supprime des blips en temps réel |
| `/drugcreator` | 🌿 Création de champs de récolte, labos de transformation et points de vente de drogues |

---

## 🎨 QS-Apparence

| Commande | Description |
|----------|-------------|
| `/save-images` | Démarre le moteur de photographie pour les vêtements |
| `/test_cam_saver [component] [componentId]` | Teste le système de sauvegarde de la caméra |
| `/create_shop` | Crée des boutiques vêtements, coiffeurs, tatoueurs ou chirurgiens |
| `/delete_shop` | Supprime une boutique créée en jeu |
| `/appearance` | Ouvre le menu de personnalisation (admin) |
| `/reloadskin` | Actualise le skin du joueur |
| `/clearstuckprops` | Corrige un accessoire bloqué sur le personnage |
| `/migrateskins` | Migration depuis qb-clothing, esx_skin ou fivem-appearance |
| `/migrateoutfits` | Compatible uniquement avec illenium-appearance |
