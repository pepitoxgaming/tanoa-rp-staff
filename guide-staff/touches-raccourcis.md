# 🎮 Touches & Raccourcis FiveM

{% hint style="info" %}
Ces touches sont les raccourcis les plus utiles en jeu pour le staff.
{% endhint %}

---

## ⌨️ Touches de base

| Touche | Action |
|--------|--------|
| `F1` | Menu principal / ESX Menu |
| `F2` | Inventaire (selon config) |
| `F3` | Animations / Emotes |
| `F5` | Menu job (selon métier) |
| `F6` | Menu métier |
| `F8` | Console FiveM (debug) |
| `F11` | Carte complète |
| `M` | Carte GPS |
| `Z` | Mains en l'air |
| `X` | S'allonger |
| `G` | Entrer/Sortir véhicule (passager) |
| `E` | Interaction / Action contextuelle |

---

## 🚗 En véhicule

| Touche | Action |
|--------|--------|
| `L` | Verrouiller/Déverrouiller |
| `Q` | Gyrophare (véhicule police/ems) |
| `R` | Sirène |
| `Num Lock` | Pilote automatique |
| `H` | Klaxon (maintenir) |

---

## 👮 Staff uniquement

| Touche | Action |
|--------|--------|
| `F4` | Menu admin (Nutty / vMenu) |
| `Insert` | NoClip ON/OFF |
| `Home` | Téléportation rapide (waypoint) |
| `PageUp/Down` | Monter/Descendre en NoClip |

{% hint style="warning" %}
Les touches peuvent varier selon la configuration du serveur. En cas de doute, utilise `/help` en jeu.
{% endhint %}
